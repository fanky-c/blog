# MyBlog

基于 `Hexo + aircloud` 的个人博客项目。

## 分支说明

- `dev/blog`：开发分支
- `release/blog`：静态产物分支
- `master`：默认分支

## 常用命令

- 启动本地服务：`npm run dev`
- 新建文章：`npx hexo new "文章标题"`
- 构建产物：`npm run build`
- 发布部署：`npm run deploy`

## 环境要求

- Node.js `>= 14`
