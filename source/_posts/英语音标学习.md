## 元音（20）

### 单元音（12）
```js
// 1、两难
[æ] --> b(a)g    [a:] --> c(ar)
[e] --> l(e)g    [ʌ] --> b(u)s

// 2、两易
[i:] --> s(ee)    [u:] --> r(u)ler
[i] --> s(i)t     [u]  --> l(oo)k

// 3、两嘴型
[ɔ:] --> n(o)rth  [ə:] --> e(ar)th
[ɔ]  --> b(o)x    [ə]  --> teach(er)
```

### 双元音（8）

**两个单元音组成就是双元音**

```js
[ai] --> sk(y)
[ei] --> n(a)me
[ɔi] --> b(oy)

[iə] --> (ear)
[eə] --> (air)
[uə] --> s(ure)

[au] --> n(ow)
[əu] --> kn(ow)
```
